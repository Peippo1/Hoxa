# Hoxa Web App

Next.js + TypeScript + Tailwind companion experience for Hoxa.

## Purpose

The web app is a polished secondary surface focused on:

- dashboard and planning visibility
- adaptive weekly calendar
- workout library browsing
- progress review
- lightweight social accountability

## Current State

- static mock-data-driven starter screens
- branded UI system aligned to the Hoxa palette
- route structure aligned to `Plan`, `Train`, `Progress`, and `Social`
- loading and error scaffolding ready for future backend integration

## Run

```bash
npm install
npm run dev
```

## Environment

Copy values from [`.env.example`](./.env.example) into a local `.env.local` if needed. Use placeholders only in the repo.
