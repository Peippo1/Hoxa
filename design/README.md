# Hoxa Design Notes

## Visual Direction

Hoxa should feel warm, editorial, calm, and premium. The interface should borrow more from hospitality and lifestyle products than from loud performance dashboards.

## Brand Colours

- Dusty rose: `#DC9B9B`
- Warm canvas: `#F6F4E8`
- Soft sage: `#E5EEE4`
- Mint wash: `#C0E1D2`

## UI Guidance

- rounded cards with generous internal padding
- soft shadows and subtle borders
- comfortable line lengths
- clear typography hierarchy
- motion used sparingly for emphasis and flow
- mobility and recovery surfaced with equal visual importance
